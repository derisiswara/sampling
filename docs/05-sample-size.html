Dari formula dasar hingga power analysis, Bayesian, dan pendekatan ML

Penentuan ukuran sampel (sample size determination) adalah keputusan desain paling kritis — terlalu kecil menghasilkan presisi rendah, terlalu besar membuang sumber daya. Bab ini menyajikan pendekatan dari yang paling sederhana hingga advanced (Cochran, 1977; Cohen, 1988; Lohr, 2021).

1 SRS untuk Estimasi Mean

1.1 Formula Dasar

Untuk mengestimasi population mean \(\bar{Y}\) dengan margin of error \(e\) pada confidence level \((1-\alpha)\):

\[ n = \frac{z^2_{1-\alpha/2} \cdot \sigma^2}{e^2} \tag{1}\]

Dengan Finite Population Correction (FPC):

\[ n = \frac{n_0}{1 + n_0/N} \quad \text{di mana } n_0 = \frac{z^2 \sigma^2}{e^2} \tag{2}\]

\(\sigma^2\) biasanya tidak diketahui. Gunakan: (1) pilot study, (2) data sebelumnya, (3) range/4 rule: \(\sigma \approx (\max - \min)/4\), atau (4) conservative estimate.

1.2 Contoh Numerik

Ingin mengestimasi mean income dengan $e = $ Rp 500.000, $= $ Rp 3.000.000, 95% confidence:

\[ n_0 = \frac{1.96^2 \times 3{,}000{,}000^2}{500{,}000^2} = \frac{3.8416 \times 9 \times 10^{12}}{2.5 \times 10^{11}} = 138.3 \approx 139 \]

Dengan FPC (\(N = 10{,}000\)): \(n = 139 / (1 + 139/10000) = 137.1 \approx 138\)

2 SRS untuk Estimasi Proporsi

2.1 Formula

\[ n = \frac{z^2 \cdot p(1-p)}{e^2} \tag{3}\]

Dengan FPC: \(n = n_0 / (1 + n_0/N)\).

TipConservative vs Informed
  • Conservative (\(p = 0.5\)): menghasilkan \(n\) terbesar → aman untuk semua proporsi.
  • Informed prior (\(p\) dari studi sebelumnya): menghasilkan \(n\) lebih kecil → lebih efisien jika prior akurat.

Contoh: \(e = 3\%\), 95% CI. Conservative: \(n = 1.96^2 \times 0.25 / 0.03^2 = 1{,}068\). Jika \(p = 0.10\): \(n = 1.96^2 \times 0.09 / 0.03^2 = 384\).

3 Perbandingan Dua Mean / Dua Proporsi

3.1 Dua Independent Means

\[ n_{\text{per group}} = \frac{(z_{1-\alpha/2} + z_{1-\beta})^2 \cdot 2\sigma^2}{\delta^2} \tag{4}\]

di mana \(\delta = \mu_1 - \mu_2\) adalah perbedaan yang ingin dideteksi.

3.2 Dua Independent Proportions

\[ n_{\text{per group}} = \frac{(z_{1-\alpha/2} + z_{1-\beta})^2 \cdot [p_1(1-p_1) + p_2(1-p_2)]}{(p_1 - p_2)^2} \tag{5}\]

4 Adjustment Factors

4.1 Non-Response Adjustment

\[ n_{adj} = \frac{n}{1 - r} \tag{6}\]

di mana \(r\) = expected non-response rate. Jika \(n = 400\) dan \(r = 0.20\): \(n_{adj} = 400 / 0.80 = 500\).

4.2 Design Effect Adjustment

\[ n_{eff} = n_{\text{SRS}} \times \text{DEFF} \tag{7}\]

4.3 Combined Adjustment

\[ n_{final} = \frac{n_{\text{SRS}} \times \text{DEFF}}{1 - r} \tag{8}\]

PeringatanJangan Lupa Adjustment!

Rumus dasar mengasumsikan SRS dan 100% response rate. Dalam praktik, DEFF = 2–3 dan non-response = 20–40% sangat umum. Sampel 400 (rumus dasar) bisa menjadi \(400 \times 2.5 / 0.70 = 1{,}429\) setelah adjustment.

5 Sample Size untuk Stratified Sampling

Untuk alokasi proporsional:

\[ n = \frac{z^2 \sum W_h S_h^2}{e^2 + \frac{z^2}{N} \sum W_h S_h^2} \]

Untuk alokasi Neyman, \(n\) lebih kecil karena alokasi lebih efisien — tetapi butuh informasi \(S_h\) yang akurat.

7 Power Analysis Framework

7.1 Konsep

Parameter Simbol Definisi
Significance level \(\alpha\) P(reject H₀ | H₀ true) — typically 0.05
Type II error rate \(\beta\) P(fail to reject H₀ | H₁ true)
Power \(1 - \beta\) P(reject H₀ | H₁ true) — typically 0.80 or 0.90
Effect size \(d\), \(h\), \(f\) Standardized magnitude of the effect

7.2 Cohen’s Conventions (dan Kritiknya)

Test Small Medium Large
Two means (\(d\)) 0.2 0.5 0.8
Correlation (\(r\)) 0.1 0.3 0.5
Proportions (\(h\)) 0.2 0.5 0.8
ANOVA (\(f\)) 0.1 0.25 0.4
PeringatanKritik Cohen’s Conventions

Cohen’s conventions adalah benchmarks generik — bukan pengganti effect size yang context-specific. “Small” effect di clinical trial bisa sangat bermakna (obat menyelamatkan nyawa). “Large” effect di social science bisa trivial. Selalu tentukan effect size berdasarkan:

  1. Minimum Detectable Effect (MDE) yang bermakna secara praktis
  2. Literature review pada domain yang sama
  3. Pilot study data

#' Comprehensive Sample Size Calculator
#'
#' @param type "mean" atau "proportion"
#' @param sigma SD populasi (untuk type = "mean")
#' @param p Expected proportion (untuk type = "proportion")
#' @param e Margin of error
#' @param alpha Significance level
#' @param N_pop Population size (Inf untuk infinite)
#' @param deff Design effect
#' @param response_rate Expected response rate (0 to 1)
#' @param n_subgroups Jumlah subgroup yang perlu dianalisis
calc_sample_size <- function(type = c("mean", "proportion"),
                              sigma = NULL, p = NULL,
                              e, alpha = 0.05,
                              N_pop = Inf, deff = 1,
                              response_rate = 1,
                              n_subgroups = 1) {
  type <- match.arg(type)
  z <- qnorm(1 - alpha / 2)

  if (type == "mean") {
    stopifnot(!is.null(sigma))
    n0 <- (z^2 * sigma^2) / e^2
  } else {
    if (is.null(p)) p <- 0.5
    n0 <- (z^2 * p * (1 - p)) / e^2
  }

  # FPC
  if (is.finite(N_pop)) {
    n0 <- n0 / (1 + n0 / N_pop)
  }

  # DEFF
  n_adj <- n0 * deff

  # Subgroups (ensure each subgroup has sufficient n)
  n_adj <- n_adj * n_subgroups

  # Non-response
  n_final <- ceiling(n_adj / response_rate)

  tibble(
    `n (basic)` = ceiling(n0),
    `n (with DEFF)` = ceiling(n0 * deff),
    `n (with subgroups)` = ceiling(n0 * deff * n_subgroups),
    `n (final, with NR adj)` = n_final,
    DEFF = deff,
    `Response Rate` = response_rate,
    `Subgroups` = n_subgroups
  )
}

import numpy as np
from scipy.stats import norm
import pandas as pd

def calc_sample_size(
    type='mean', sigma=None, p=None, e=None,
    alpha=0.05, N_pop=np.inf, deff=1,
    response_rate=1, n_subgroups=1
):
    z = norm.ppf(1 - alpha / 2)

    if type == 'mean':
        n0 = (z**2 * sigma**2) / e**2
    else:
        if p is None: p = 0.5
        n0 = (z**2 * p * (1 - p)) / e**2

    if np.isfinite(N_pop):
        n0 = n0 / (1 + n0 / N_pop)

    n_adj = n0 * deff * n_subgroups
    n_final = int(np.ceil(n_adj / response_rate))

    return pd.DataFrame({
        'Parameter': ['n (basic)', 'n (with DEFF)',
                      'n (final, with NR adj)', 'DEFF', 'Response Rate'],
        'Value': [int(np.ceil(n0)), int(np.ceil(n0 * deff)),
                  n_final, deff, response_rate]
    })

10 Bayesian Sample Size Determination

10.1 Average Posterior Variance Criterion (APVC)

Pilih \(n\) yang meminimalkan expected posterior variance:

\[ n^* = \arg\min_n \; E_y\left[\text{Var}(\theta \mid y)\right] \]

Untuk normal model dengan prior \(\theta \sim N(\mu_0, \tau^2)\) dan data \(y \sim N(\theta, \sigma^2/n)\):

\[ \text{Var}(\theta \mid y) = \frac{1}{1/\tau^2 + n/\sigma^2} \tag{11}\]

10.2 Average Coverage Criterion (ACC)

Pilih \(n\) sehingga posterior credible interval memiliki average coverage ≥ target:

\[ n^* = \min\{n : E_y[P(\theta \in \text{CI}_{1-\alpha} \mid y)] \geq 1 - \alpha\} \]

10.3 Perbandingan Frequentist vs Bayesian

Aspek Frequentist Bayesian
Informasi prior Tidak digunakan Diinkorporasi via prior
Kriteria MoE, power Posterior variance, coverage
Interpretasi Long-run frequency Direct probability
Sensitivity Terhadap σ, effect size Terhadap prior + σ
Advantage Simpler, well-understood Can incorporate prior knowledge

11.1 Learning Curves Approach

Alih-alih formula, gunakan learning curves — plot performance metric vs training set size:

  1. Train model pada subset data dengan ukuran berbeda: \(n_1 < n_2 < \ldots < n_k\)
  2. Evaluate pada hold-out test set
  3. Fit inverse power law: \(\text{Error}(n) = an^{-b} + c\)

11.2 Key Takeaways

  1. Formula dasar adalah starting point — selalu adjust untuk DEFF, non-response, dan subgroup analysis.
  2. Effect size harus context-driven, bukan mengikuti Cohen’s conventions secara membabi-buta.
  3. Power analysis membutuhkan empat parameter (\(\alpha\), power, effect size, \(n\)) — fix tiga, solve satu.
  4. Bayesian sample size memanfaatkan prior information dan sering menghasilkan \(n\) lebih kecil jika prior informatif.
  5. Untuk ML, gunakan learning curves daripada formula — lebih realistis dan data-driven.

## SRS untuk Estimasi Mean

### Formula Dasar

Untuk mengestimasi population mean $\bar{Y}$ dengan margin of error $e$ pada confidence level $(1-\alpha)$:

$$
n = \frac{z^2_{1-\alpha/2} \cdot \sigma^2}{e^2}
$$ {#eq-n-mean}

Dengan Finite Population Correction (FPC):

$$
n = \frac{n_0}{1 + n_0/N} \quad \text{di mana } n_0 = \frac{z^2 \sigma^2}{e^2}
$$ {#eq-n-mean-fpc}

::: {.column-margin}
$\sigma^2$ biasanya tidak diketahui. Gunakan: (1) pilot study, (2) data sebelumnya, (3) range/4 rule: $\sigma \approx (\max - \min)/4$, atau (4) conservative estimate.
:::

### Contoh Numerik

Ingin mengestimasi mean income dengan $e = $ Rp 500.000, $\sigma = $ Rp 3.000.000, 95% confidence:

$$
n_0 = \frac{1.96^2 \times 3{,}000{,}000^2}{500{,}000^2} = \frac{3.8416 \times 9 \times 10^{12}}{2.5 \times 10^{11}} = 138.3 \approx 139
$$

Dengan FPC ($N = 10{,}000$): $n = 139 / (1 + 139/10000) = 137.1 \approx 138$

## SRS untuk Estimasi Proporsi

### Formula

$$
n = \frac{z^2 \cdot p(1-p)}{e^2}
$$ {#eq-n-prop}

Dengan FPC: $n = n_0 / (1 + n_0/N)$.

::: {.callout-tip}
## Conservative vs Informed

- **Conservative** ($p = 0.5$): menghasilkan $n$ terbesar → aman untuk semua proporsi.
- **Informed prior** ($p$ dari studi sebelumnya): menghasilkan $n$ lebih kecil → lebih efisien jika prior akurat.

Contoh: $e = 3\%$, 95% CI. Conservative: $n = 1.96^2 \times 0.25 / 0.03^2 = 1{,}068$. Jika $p = 0.10$: $n = 1.96^2 \times 0.09 / 0.03^2 = 384$.
:::

## Perbandingan Dua Mean / Dua Proporsi

### Dua Independent Means

$$
n_{\text{per group}} = \frac{(z_{1-\alpha/2} + z_{1-\beta})^2 \cdot 2\sigma^2}{\delta^2}
$$ {#eq-n-two-means}

di mana $\delta = \mu_1 - \mu_2$ adalah perbedaan yang ingin dideteksi.

### Dua Independent Proportions

$$
n_{\text{per group}} = \frac{(z_{1-\alpha/2} + z_{1-\beta})^2 \cdot [p_1(1-p_1) + p_2(1-p_2)]}{(p_1 - p_2)^2}
$$ {#eq-n-two-props}

## Adjustment Factors

### Non-Response Adjustment

$$
n_{adj} = \frac{n}{1 - r}
$$ {#eq-n-nonresponse}

di mana $r$ = expected non-response rate. Jika $n = 400$ dan $r = 0.20$: $n_{adj} = 400 / 0.80 = 500$.

### Design Effect Adjustment

$$
n_{eff} = n_{\text{SRS}} \times \text{DEFF}
$$ {#eq-n-deff}

### Combined Adjustment

$$
n_{final} = \frac{n_{\text{SRS}} \times \text{DEFF}}{1 - r}
$$ {#eq-n-combined}

::: {.callout-warning}
## Jangan Lupa Adjustment!
Rumus dasar mengasumsikan SRS dan 100% response rate. Dalam praktik, DEFF = 2–3 dan non-response = 20–40% sangat umum. Sampel 400 (rumus dasar) bisa menjadi $400 \times 2.5 / 0.70 = 1{,}429$ setelah adjustment.
:::

## Sample Size untuk Stratified Sampling

Untuk alokasi proporsional:

$$
n = \frac{z^2 \sum W_h S_h^2}{e^2 + \frac{z^2}{N} \sum W_h S_h^2}
$$

Untuk alokasi Neyman, $n$ lebih kecil karena alokasi lebih efisien — tetapi butuh informasi $S_h$ yang akurat.

## Power Analysis Framework

### Konsep

| Parameter | Simbol | Definisi |
|-----------|--------|----------|
| Significance level | $\alpha$ | P(reject H₀ \| H₀ true) — typically 0.05 |
| Type II error rate | $\beta$ | P(fail to reject H₀ \| H₁ true) |
| Power | $1 - \beta$ | P(reject H₀ \| H₁ true) — typically 0.80 or 0.90 |
| Effect size | $d$, $h$, $f$ | Standardized magnitude of the effect |

### Cohen's Conventions (dan Kritiknya)

| Test | Small | Medium | Large |
|------|-------|--------|-------|
| Two means ($d$) | 0.2 | 0.5 | 0.8 |
| Correlation ($r$) | 0.1 | 0.3 | 0.5 |
| Proportions ($h$) | 0.2 | 0.5 | 0.8 |
| ANOVA ($f$) | 0.1 | 0.25 | 0.4 |

::: {.callout-warning}
## Kritik Cohen's Conventions
Cohen's conventions adalah **benchmarks generik** — bukan pengganti effect size yang context-specific. "Small" effect di clinical trial bisa sangat bermakna (obat menyelamatkan nyawa). "Large" effect di social science bisa trivial. **Selalu tentukan effect size berdasarkan:**

1. Minimum Detectable Effect (MDE) yang bermakna secara praktis
2. Literature review pada domain yang sama
3. Pilot study data
:::

```{r}
#' Comprehensive Sample Size Calculator
#'
#' @param type "mean" atau "proportion"
#' @param sigma SD populasi (untuk type = "mean")
#' @param p Expected proportion (untuk type = "proportion")
#' @param e Margin of error
#' @param alpha Significance level
#' @param N_pop Population size (Inf untuk infinite)
#' @param deff Design effect
#' @param response_rate Expected response rate (0 to 1)
#' @param n_subgroups Jumlah subgroup yang perlu dianalisis
calc_sample_size <- function(type = c("mean", "proportion"),
                              sigma = NULL, p = NULL,
                              e, alpha = 0.05,
                              N_pop = Inf, deff = 1,
                              response_rate = 1,
                              n_subgroups = 1) {
  type <- match.arg(type)
  z <- qnorm(1 - alpha / 2)

  if (type == "mean") {
    stopifnot(!is.null(sigma))
    n0 <- (z^2 * sigma^2) / e^2
  } else {
    if (is.null(p)) p <- 0.5
    n0 <- (z^2 * p * (1 - p)) / e^2
  }

  # FPC
  if (is.finite(N_pop)) {
    n0 <- n0 / (1 + n0 / N_pop)
  }

  # DEFF
  n_adj <- n0 * deff

  # Subgroups (ensure each subgroup has sufficient n)
  n_adj <- n_adj * n_subgroups

  # Non-response
  n_final <- ceiling(n_adj / response_rate)

  tibble(
    `n (basic)` = ceiling(n0),
    `n (with DEFF)` = ceiling(n0 * deff),
    `n (with subgroups)` = ceiling(n0 * deff * n_subgroups),
    `n (final, with NR adj)` = n_final,
    DEFF = deff,
    `Response Rate` = response_rate,
    `Subgroups` = n_subgroups
  )
}
```

## Multi-Objective Sample Size

Ketika survei mengukur banyak variabel, hitung $n$ untuk **setiap variabel kunci**, lalu ambil **maksimum**:

$$
n^* = \max(n_1, n_2, \ldots, n_k)
$$ {#eq-n-multi}

::: {.panel-tabset}

## Bayesian Sample Size Determination

### Average Posterior Variance Criterion (APVC)

Pilih $n$ yang meminimalkan expected posterior variance:

$$
n^* = \arg\min_n \; E_y\left[\text{Var}(\theta \mid y)\right]
$$

Untuk normal model dengan prior $\theta \sim N(\mu_0, \tau^2)$ dan data $y \sim N(\theta, \sigma^2/n)$:

$$
\text{Var}(\theta \mid y) = \frac{1}{1/\tau^2 + n/\sigma^2}
$$ {#eq-bayesian-posterior-var}

### Average Coverage Criterion (ACC)

Pilih $n$ sehingga posterior credible interval memiliki average coverage ≥ target:

$$
n^* = \min\{n : E_y[P(\theta \in \text{CI}_{1-\alpha} \mid y)] \geq 1 - \alpha\}
$$

### Perbandingan Frequentist vs Bayesian

| Aspek | Frequentist | Bayesian |
|-------|-------------|----------|
| **Informasi prior** | Tidak digunakan | Diinkorporasi via prior |
| **Kriteria** | MoE, power | Posterior variance, coverage |
| **Interpretasi** | Long-run frequency | Direct probability |
| **Sensitivity** | Terhadap σ, effect size | Terhadap prior + σ |
| **Advantage** | Simpler, well-understood | Can incorporate prior knowledge |

::: {.panel-tabset}

### R

## Sample Size untuk Machine Learning

### Learning Curves Approach

Alih-alih formula, gunakan **learning curves** — plot performance metric vs training set size:

1. Train model pada subset data dengan ukuran berbeda: $n_1 < n_2 < \ldots < n_k$
2. Evaluate pada hold-out test set
3. Fit inverse power law: $\text{Error}(n) = an^{-b} + c$
4. Extrapolate untuk menentukan $n$ yang mencapai target performance

::: {.panel-tabset}

### R

```{r}
#| label: fig-learning-curve
#| fig-cap: "Learning curve: performance meningkat dengan n tapi dengan diminishing returns."
#| fig-width: 9
#| fig-height: 5

set.seed(2024)

# Simulate learning curve data
n_train_sizes <- c(50, 100, 200, 500, 1000, 2000, 3000, 5000, 7000)

# Simulate: error = a * n^(-b) + c + noise
a <- 5; b <- 0.5; c <- 0.15
lc_data <- tibble(
  n_train = n_train_sizes,
  error = a * n_train_sizes^(-b) + c + rnorm(length(n_train_sizes), 0, 0.01)
)

# Fit power law
fit <- nls(error ~ a * n_train^(-b) + c, data = lc_data,
           start = list(a = 5, b = 0.5, c = 0.15))

# Predict
n_pred <- seq(50, 10000, by = 50)
pred_error <- predict(fit, newdata = tibble(n_train = n_pred))

target_error <- 0.20
n_for_target <- n_pred[which.min(abs(pred_error - target_error))]

ggplot(lc_data, aes(x = n_train, y = error)) +
  geom_point(size = 3, color = "#2563eb") +
  geom_line(data = tibble(n_train = n_pred, error = pred_error),
            color = "#dc2626", linewidth = 1) +
  geom_hline(yintercept = target_error, linetype = "dashed",
             color = "#059669") +
  geom_vline(xintercept = n_for_target, linetype = "dotted",
             color = "#059669") +
  annotate("text", x = n_for_target + 500, y = target_error + 0.01,
           label = sprintf("n ≈ %d for error = %.2f", n_for_target, target_error),
           color = "#059669", size = 3.5) +
  scale_x_continuous(labels = label_comma()) +
  labs(
    title = "Learning Curve: Sample Size for ML",
    subtitle = "Fitted inverse power law: Error = a·n^(-b) + c",
    x = "Training Set Size",
    y = "Test Error (e.g., 1 - Accuracy)"
  )
```

```{python}
import numpy as np
import matplotlib.pyplot as plt
from scipy.optimize import curve_fit

np.random.seed(2024)

n_sizes = np.array([50, 100, 200, 500, 1000, 2000, 3000, 5000, 7000])
errors = 5 * n_sizes**(-0.5) + 0.15 + np.random.normal(0, 0.01, len(n_sizes))

def power_law(n, a, b, c):
    return a * n**(-b) + c

popt, _ = curve_fit(power_law, n_sizes, errors, p0=[5, 0.5, 0.15])

n_pred = np.arange(50, 10001, 50)
pred_errors = power_law(n_pred, *popt)

target = 0.20
n_target = n_pred[np.argmin(np.abs(pred_errors - target))]

fig, ax = plt.subplots(figsize=(9, 5))
ax.scatter(n_sizes, errors, s=60, color='#2563eb', zorder=5)
ax.plot(n_pred, pred_errors, color='#dc2626', linewidth=1.5)
ax.axhline(target, linestyle='--', color='#059669')
ax.axvline(n_target, linestyle=':', color='#059669')
ax.annotate(f'n ≈ {n_target} for error = {target}',
            xy=(n_target, target), xytext=(n_target + 1000, target + 0.03),
            arrowprops=dict(arrowstyle='->', color='#059669'),
            color='#059669')
ax.set_xlabel('Training Set Size')
ax.set_ylabel('Test Error')
ax.set_title('Learning Curve: ML Sample Size', fontweight='bold')
plt.tight_layout()
plt.show()
```

### Key Takeaways

1. **Formula dasar adalah starting point** — selalu adjust untuk DEFF, non-response, dan subgroup analysis.
2. **Effect size harus context-driven**, bukan mengikuti Cohen's conventions secara membabi-buta.
3. **Power analysis** membutuhkan empat parameter ($\alpha$, power, effect size, $n$) — fix tiga, solve satu.
4. **Bayesian sample size** memanfaatkan prior information dan sering menghasilkan $n$ lebih kecil jika prior informatif.
5. **Untuk ML**, gunakan learning curves daripada formula — lebih realistis dan data-driven.
