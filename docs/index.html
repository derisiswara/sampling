Prerequisites

CatatanApa yang perlu dikuasai sebelum memulai?
  • Statistika dasar: distribusi, mean, varians, confidence interval, hypothesis testing.
  • R atau Python: kemampuan dasar manipulasi data dan visualisasi. Materi menyediakan code dalam kedua bahasa.
  • Konsep probabilitas: pemahaman dasar teori probabilitas dan distribusi sampling.
  • Linear algebra (opsional): membantu untuk memahami derivasi pada bab lanjut.
TipKapan teknik sampling penting vs. bisa diabaikan?

Sampling PENTING ketika:

  • Populasi terlalu besar/mahal untuk diteliti seluruhnya (sensus)
  • Pengujian bersifat destruktif (quality control)
  • Membutuhkan hasil cepat dengan sumber daya terbatas
  • Perlu estimasi parameter dengan presisi yang terukur

Sampling BISA DIABAIKAN ketika:

  • Populasi kecil dan mudah dijangkau → lakukan sensus
  • Data sudah tersedia secara lengkap (administrative data, log data)
  • Analisis big data di mana seluruh data bisa diproses
  • Tujuan hanya eksplorasi/deskriptif tanpa perlu inferensi ke populasi

Seluruh materi menggunakan notasi konsisten berikut:

Simbol Makna
\(N\) Ukuran populasi
\(n\) Ukuran sampel
\(Y_i\) Nilai variabel untuk unit ke-\(i\)
\(\bar{Y}\) Population mean
\(\bar{y}\) Sample mean
\(S^2\) Population variance
\(s^2\) Sample variance
\(\hat{\theta}\) Estimator dari parameter \(\theta\)
\(\text{Var}(\hat{\theta})\) Variance of estimator
\(\text{MSE}(\hat{\theta})\) Mean Squared Error
\(\text{DEFF}\) Design Effect
\(f = n/N\) Sampling fraction
