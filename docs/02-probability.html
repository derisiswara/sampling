SRS, Systematic, Stratified, Cluster, dan PPS — teori, estimator, dan implementasi

Probability sampling adalah satu-satunya paradigma sampling yang memungkinkan inferensi berbasis desain (design-based inference) — setiap unit memiliki probabilitas seleksi yang diketahui dan >0, sehingga kita dapat mengkuantifikasi ketidakpastian estimasi secara formal (Lohr, 2021).

1 Simple Random Sampling (SRS)

1.1 Konsep

Setiap subset berukuran \(n\) dari populasi \(N\) memiliki peluang yang sama untuk terpilih. Probabilitas inklusi setiap unit: \(\pi_i = n/N\).

CatatanSRS With vs. Without Replacement
  • SRSWR (with replacement): unit bisa terpilih >1 kali. Observasi independen. Jarang digunakan dalam survei.
  • SRSWOR (without replacement): unit hanya terpilih sekali. Standar dalam survei. Observasi tidak independen tetapi lebih efisien.

1.2 Estimator

Untuk SRSWOR:

Parameter Estimator Variance Estimator
Mean \(\bar{Y}\) \(\bar{y} = \frac{1}{n}\sum_{i=1}^n y_i\) \(\hat{V}(\bar{y}) = (1-f)\frac{s^2}{n}\)
Total \(Y_{total}\) \(\hat{Y} = N\bar{y}\) \(\hat{V}(\hat{Y}) = N^2(1-f)\frac{s^2}{n}\)
Proporsi \(P\) \(\hat{p} = \frac{1}{n}\sum_{i=1}^n y_i\) \(\hat{V}(\hat{p}) = (1-f)\frac{\hat{p}(1-\hat{p})}{n-1}\)

di mana \(f = n/N\) adalah sampling fraction dan \(s^2 = \frac{1}{n-1}\sum(y_i - \bar{y})^2\).

1.3 Finite Population Correction (FPC)

\[ \text{FPC} = 1 - f = 1 - \frac{n}{N} \tag{1}\]

FPC signifikan ketika \(f > 0.05\) (sampling lebih dari 5% populasi). Untuk populasi besar atau sampling fraction kecil, FPC ≈ 1 dan bisa diabaikan.

2 Systematic Sampling

2.1 Konsep

Pilih unit setiap interval \(k = \lfloor N/n \rfloor\) setelah random start \(r \in [1, k]\). Sampel: unit ke-\(r, r+k, r+2k, \ldots\)

2.2 Risiko dan Mitigasi

PeringatanPeriodicity Danger

Jika frame memiliki pola periodik dengan periode = \(k\) (atau kelipatan), systematic sampling menghasilkan estimasi yang sangat biased. Contoh: daftar rumah tangga yang diurutkan per blok (setiap 10 rumah = 1 RT), lalu \(k = 10\) → selalu memilih posisi yang sama di setiap RT.

  • Mitigasi: (1) randomize urutan frame sebelum sampling, (2) gunakan circular systematic sampling, (3) multiple random starts.

3 Stratified Random Sampling

3.1 Prinsip

Bagi populasi menjadi \(H\) strata yang homogen secara internal (intra-strata variance kecil) dan heterogen antar strata (inter-strata variance besar). Tarik SRS independen dari setiap stratum.

3.2 Estimator

\[ \bar{y}_{st} = \sum_{h=1}^{H} W_h \bar{y}_h \tag{2}\]

di mana \(W_h = N_h / N\) adalah bobot stratum.

\[ \text{Var}(\bar{y}_{st}) = \sum_{h=1}^{H} W_h^2 \left(1 - f_h\right) \frac{s_h^2}{n_h} \tag{3}\]

3.3 Metode Alokasi

Alokasi Formula \(n_h\) Kapan Optimal
Proportional \(n_h = n \cdot W_h\) Default; varians antar strata mirip
Neyman (Optimal) \(n_h = n \cdot \frac{W_h S_h}{\sum W_h S_h}\) Varians berbeda antar strata
Power \(n_h = n \cdot \frac{W_h S_h^a}{\sum W_h S_h^a}\) Compromise (\(a=0.5\) populer)

Neyman allocation meminimalkan \(\text{Var}(\bar{y}_{st})\) untuk total \(n\) tetap. Alokasi proporsional lebih robust jika survei mengukur banyak variabel.

Kode 
set.seed(2024)
n_strat <- 200

# Proportional allocation
strata_info <- pop |>
  summarise(
    N_h = n(),
    S_h = sd(income),
    mean_h = mean(income),
    .by = region
  ) |>
  mutate(
    W_h = N_h / N,
    n_prop = round(n_strat * W_h),
    n_neyman = round(n_strat * (W_h * S_h) / sum(W_h * S_h))
  )

# Draw proportional stratified sample
strat_sample <- pop |>
  left_join(strata_info |> select(region, n_prop), by = "region") |>
  group_by(region, n_prop) |>
  group_modify(~ slice_sample(.x, n = min(.y$n_prop[[1]], nrow(.x)))) |>
  ungroup() |>
  select(-n_prop)

# Compute stratified mean
strat_estimates <- strat_sample |>
  summarise(
    ybar_h = mean(income),
    s2_h = var(income),
    n_h = n(),
    .by = region
  ) |>
  left_join(strata_info |> select(region, W_h, N_h), by = "region") |>
  mutate(f_h = n_h / N_h)

ybar_st <- sum(strat_estimates$W_h * strat_estimates$ybar_h)
var_st <- sum(strat_estimates$W_h^2 * (1 - strat_estimates$f_h) *
                strat_estimates$s2_h / strat_estimates$n_h)
se_st <- sqrt(var_st)

# Plot
ggplot(pop, aes(x = income, fill = region)) +
  geom_histogram(bins = 40, alpha = 0.4, position = "identity") +
  geom_histogram(data = strat_sample, bins = 30, alpha = 0.8) +
  facet_wrap(~region, scales = "free_y") +
  scale_x_continuous(labels = label_comma()) +

    title = "Stratified Sampling by Region",

                       n_strat, format(round(ybar_st), big.mark = ","),

4 Cluster Sampling

4.1 Konsep

Populasi dibagi menjadi cluster (kelompok, biasanya geografis). Pilih subset cluster secara random, lalu observasi semua unit dalam cluster terpilih (one-stage) atau subsample dalam cluster (two-stage).

4.2 Intra-class Correlation Coefficient (ICC)

\[ \rho = \frac{\sigma^2_b}{\sigma^2_b + \sigma^2_w} \tag{4}\]

di mana \(\sigma^2_b\) = between-cluster variance, \(\sigma^2_w\) = within-cluster variance.

ICC tinggi → unit dalam cluster mirip satu sama lain → cluster sampling kurang efisien dibanding SRS. Idealnya, cluster harus heterogen secara internal (ICC rendah).

4.3 Design Effect (DEFF) karena Clustering

\[ \text{DEFF} = 1 + (m - 1)\rho \tag{5}\]

di mana \(m\) = rata-rata jumlah unit per cluster. Jika \(\rho = 0.05\) dan \(m = 20\), maka \(\text{DEFF} = 1 + 19 \times 0.05 = 1.95\) — hampir dua kali lipat varians dibanding SRS!

5 Probability Proportional to Size (PPS)

5.1 Konsep

Unit dipilih dengan probabilitas proporsional terhadap ukuran (size measure) — misalnya populasi, jumlah rumah tangga, atau revenue. Unit besar memiliki peluang lebih tinggi untuk terpilih.

5.2 Estimator

Horvitz-Thompson estimator (Horvitz & Thompson, 1952):

\[ \hat{Y}_{HT} = \sum_{i \in s} \frac{y_i}{\pi_i} \tag{6}\]

di mana \(\pi_i\) = probabilitas inklusi unit \(i\).

Hansen-Hurwitz estimator (PPS with replacement) (Hansen & Hurwitz, 1943):

\[ \hat{Y}_{HH} = \frac{1}{n} \sum_{i=1}^{n} \frac{y_i}{p_i} \tag{7}\]

di mana \(p_i = X_i / X_{total}\) adalah probabilitas seleksi pada setiap draw.

7.1 Key Takeaways

  1. SRS adalah baseline — metode lain dibandingkan terhadapnya via Design Effect (DEFF).
  2. Stratified sampling selalu lebih efisien (atau sama) dengan SRS jika stratifikasi variabel berkorelasi dengan variabel survei.
  3. Cluster sampling mengorbankan presisi untuk efisiensi logistik. ICC dan cluster size menentukan seberapa besar “harga” yang dibayar.
  4. PPS sampling tepat ketika cluster berukuran berbeda-beda — memberikan self-weighting design.
  5. Pemilihan metode harus mempertimbangkan: ketersediaan frame, budget, variabilitas populasi, dan kebutuhan estimasi subgroup.

Probability sampling adalah satu-satunya paradigma sampling yang memungkinkan **inferensi berbasis desain** (*design-based inference*) — setiap unit memiliki probabilitas seleksi yang diketahui dan >0, sehingga kita dapat mengkuantifikasi ketidakpastian estimasi secara formal [@lohr2021].

## Simple Random Sampling (SRS)

### Konsep

Setiap subset berukuran $n$ dari populasi $N$ memiliki peluang yang sama untuk terpilih. Probabilitas inklusi setiap unit: $\pi_i = n/N$.

::: {.callout-note}
## SRS With vs. Without Replacement

- **SRSWR** (with replacement): unit bisa terpilih >1 kali. Observasi independen. Jarang digunakan dalam survei.
- **SRSWOR** (without replacement): unit hanya terpilih sekali. Standar dalam survei. Observasi tidak independen tetapi lebih efisien.
:::

### Estimator

Untuk SRSWOR:

| Parameter | Estimator | Variance Estimator |
|-----------|-----------|-------------------|
| Mean $\bar{Y}$ | $\bar{y} = \frac{1}{n}\sum_{i=1}^n y_i$ | $\hat{V}(\bar{y}) = (1-f)\frac{s^2}{n}$ |
| Total $Y_{total}$ | $\hat{Y} = N\bar{y}$ | $\hat{V}(\hat{Y}) = N^2(1-f)\frac{s^2}{n}$ |
| Proporsi $P$ | $\hat{p} = \frac{1}{n}\sum_{i=1}^n y_i$ | $\hat{V}(\hat{p}) = (1-f)\frac{\hat{p}(1-\hat{p})}{n-1}$ |

di mana $f = n/N$ adalah **sampling fraction** dan $s^2 = \frac{1}{n-1}\sum(y_i - \bar{y})^2$.

### Finite Population Correction (FPC)

$$
\text{FPC} = 1 - f = 1 - \frac{n}{N}
$$ {#eq-fpc}

::: {.column-margin}
FPC signifikan ketika $f > 0.05$ (sampling lebih dari 5% populasi). Untuk populasi besar atau sampling fraction kecil, FPC ≈ 1 dan bisa diabaikan.
:::

## Systematic Sampling

### Konsep

Pilih unit setiap interval $k = \lfloor N/n \rfloor$ setelah random start $r \in [1, k]$. Sampel: unit ke-$r, r+k, r+2k, \ldots$

### Risiko dan Mitigasi

::: {.callout-warning}
## Periodicity Danger
Jika frame memiliki pola periodik dengan periode = $k$ (atau kelipatan), systematic sampling menghasilkan estimasi yang sangat biased. Contoh: daftar rumah tangga yang diurutkan per blok (setiap 10 rumah = 1 RT), lalu $k = 10$ → selalu memilih posisi yang sama di setiap RT.
:::

- **Mitigasi:** (1) randomize urutan frame sebelum sampling, (2) gunakan circular systematic sampling, (3) multiple random starts.

## Stratified Random Sampling

### Prinsip

Bagi populasi menjadi $H$ strata yang **homogen secara internal** (intra-strata variance kecil) dan **heterogen antar strata** (inter-strata variance besar). Tarik SRS independen dari setiap stratum.

### Estimator

$$
\bar{y}_{st} = \sum_{h=1}^{H} W_h \bar{y}_h
$$ {#eq-strat-mean}

di mana $W_h = N_h / N$ adalah bobot stratum.

$$
\text{Var}(\bar{y}_{st}) = \sum_{h=1}^{H} W_h^2 \left(1 - f_h\right) \frac{s_h^2}{n_h}
$$ {#eq-strat-var}

### Metode Alokasi

| Alokasi | Formula $n_h$ | Kapan Optimal |
|---------|--------------|---------------|
| **Proportional** | $n_h = n \cdot W_h$ | Default; varians antar strata mirip |
| **Neyman (Optimal)** | $n_h = n \cdot \frac{W_h S_h}{\sum W_h S_h}$ | Varians berbeda antar strata |
| **Power** | $n_h = n \cdot \frac{W_h S_h^a}{\sum W_h S_h^a}$ | Compromise ($a=0.5$ populer) |

::: {.column-margin}
Neyman allocation meminimalkan $\text{Var}(\bar{y}_{st})$ untuk total $n$ tetap. Alokasi proporsional lebih robust jika survei mengukur banyak variabel.
:::

## Cluster Sampling

### Konsep

Populasi dibagi menjadi **cluster** (kelompok, biasanya geografis). Pilih subset cluster secara random, lalu observasi **semua** unit dalam cluster terpilih (one-stage) atau subsample dalam cluster (two-stage).

### Intra-class Correlation Coefficient (ICC)

$$
\rho = \frac{\sigma^2_b}{\sigma^2_b + \sigma^2_w}
$$ {#eq-icc}

di mana $\sigma^2_b$ = between-cluster variance, $\sigma^2_w$ = within-cluster variance.

::: {.column-margin}
ICC tinggi → unit dalam cluster mirip satu sama lain → cluster sampling **kurang efisien** dibanding SRS. Idealnya, cluster harus heterogen secara internal (ICC rendah).
:::

### Design Effect (DEFF) karena Clustering

$$
\text{DEFF} = 1 + (m - 1)\rho
$$ {#eq-deff-cluster}

di mana $m$ = rata-rata jumlah unit per cluster. Jika $\rho = 0.05$ dan $m = 20$, maka $\text{DEFF} = 1 + 19 \times 0.05 = 1.95$ — hampir dua kali lipat varians dibanding SRS!

## Probability Proportional to Size (PPS)

### Konsep

Unit dipilih dengan probabilitas proporsional terhadap ukuran (size measure) — misalnya populasi, jumlah rumah tangga, atau revenue. Unit besar memiliki peluang lebih tinggi untuk terpilih.

### Estimator

**Horvitz-Thompson estimator** [@horvitz1952]:

$$
\hat{Y}_{HT} = \sum_{i \in s} \frac{y_i}{\pi_i}
$$ {#eq-ht}

di mana $\pi_i$ = probabilitas inklusi unit $i$.

**Hansen-Hurwitz estimator** (PPS with replacement) [@hansen1943]:

$$
\hat{Y}_{HH} = \frac{1}{n} \sum_{i=1}^{n} \frac{y_i}{p_i}
$$ {#eq-hh}

di mana $p_i = X_i / X_{total}$ adalah probabilitas seleksi pada setiap draw.

## Tabel Ringkasan

| Metode | Estimator Mean | Variance | Kapan Digunakan | DEFF vs SRS |
|--------|---------------|----------|-----------------|-------------|
| **SRS** | $\bar{y}$ | $(1-f)\frac{s^2}{n}$ | Frame lengkap, populasi homogen | 1.0 (baseline) |
| **Systematic** | $\bar{y}_{sys}$ | ≈ SRS jika random order | Frame ordered, butuh simplicity | ≈ 1.0 |
| **Stratified** | $\sum W_h \bar{y}_h$ | $\sum W_h^2(1-f_h)\frac{s_h^2}{n_h}$ | Populasi heterogen, info auxiliary | < 1.0 (lebih efisien) |
| **Cluster** | $\frac{1}{m}\sum \bar{y}_i$ | Depends on ICC | Constraint geografis/logistik | > 1.0 (kurang efisien) |
| **PPS** | $\hat{Y}_{HT} = \sum \frac{y_i}{\pi_i}$ | HT variance | Cluster size varies, need proportional representation | Depends on correlation $y$ & size |

::: {.takeaways}
### Key Takeaways

1. **SRS adalah baseline** — metode lain dibandingkan terhadapnya via Design Effect (DEFF).
2. **Stratified sampling selalu lebih efisien** (atau sama) dengan SRS jika stratifikasi variabel berkorelasi dengan variabel survei.
3. **Cluster sampling mengorbankan presisi** untuk efisiensi logistik. ICC dan cluster size menentukan seberapa besar "harga" yang dibayar.
4. **PPS sampling** tepat ketika cluster berukuran berbeda-beda — memberikan self-weighting design.
5. **Pemilihan metode** harus mempertimbangkan: ketersediaan frame, budget, variabilitas populasi, dan kebutuhan estimasi subgroup.
